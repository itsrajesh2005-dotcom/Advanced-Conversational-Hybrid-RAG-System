1. INTRODUCTION TO PYTHON
======================================================================

Python is a high-level, interpreted, and general-purpose programming language.
It is widely used in web development, automation, data science, machine learning,
AI, cybersecurity, and software development.

Features:
- Easy to learn
- Simple syntax
- Cross-platform
- Huge library ecosystem
- Open source

Example:
print("Hello World")

======================================================================
2. INSTALLING PYTHON
======================================================================

Steps:
1. Download Python from python.org
2. Install Python
3. Add Python to PATH
4. Verify installation

Command:
python --version

======================================================================
3. PYTHON SYNTAX
======================================================================

Python uses indentation instead of braces.

Example:
if 5 > 2:
    print("Five is greater")

Rules:
- Use indentation
- Case-sensitive language
- Statements end with newline

7. CONDITIONAL STATEMENTS
======================================================================

if statement:
if x > 0:
    print("Positive")

if-else:
if x > 0:
    print("Positive")
else:
    print("Negative")

if-elif-else:
if marks >= 90:
    print("A")
elif marks >= 70:
    print("B")
else:
    print("C")

8. LOOPS
======================================================================

for loop:
for i in range(5):
    print(i)

while loop:
count = 0
while count < 5:
    print(count)
    count += 1

Loop control:
- break
- continue
- pass
